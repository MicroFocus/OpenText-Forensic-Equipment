# OpenText Forensic Equipment (formerly Tableau Hardware)

This repository contains supplemental materials for use with OTFE Imagers, Duplicators, and Bridges.

## Tree Hashing
The [tree-hashing](tree-hashing/README.md) folder contains information on the new tree hashing format supported by the TX2 Imager.

## Hash Database
The [hash-database](hash-database/README.md) folder contains information on hash database file format and usage on the TX2 Imager.
